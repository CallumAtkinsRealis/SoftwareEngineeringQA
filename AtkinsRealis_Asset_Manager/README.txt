# AtkinsRÃ©alis Asset Manager

Atkinsrealis Asset Manager is a web application designed to streamline 
asset booking and management processes. It provides a centralized platform 
for users to book assets and efficiently manage them in one place. Users 
can create accounts to facilitate the creation of new bookings and access 
various features of the application.

## Features

- **Asset Booking**: Users can easily book assets through the application.
- **Asset Management**: Efficiently manage all assets in one centralized platform.
- **User Management**: Users can create accounts to access the booking system and manage their bookings.
- **Customization**: Customizable features to tailor the application to specific user needs.

## Installation

1. Clone this repository to your local machine:
    ```bash
    git clone https://github.com/your-username/your-repository.git

2. Set up a virtual environment (recommended) and activate it:
    python3 -m venv venv
    venv\Scripts\activate.ps1

3. Navigate to project directory and install dependenancies from requirements.txt:
    - cd Atkinsrealis_Asset_Manager
    - pip install -r requirements.txt

## Usage

Once installed, you can run the Atkinsrealis Asset Manager with the following command:
- python manage.py runserver

This will start the development server, and you can access the application by navigating to http://127.0.0.1:8000 in your web browser.
